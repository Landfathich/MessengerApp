A simple, encrypted messenger for friends built with Django and PWA technology. Designed to work reliably even in challenging network environments.

## Features

- Real-time messaging using WebSockets
- End-to-End Encryption (in development)
- Progressive Web App for cross-platform compatibility
- No app store installation required
- Censorship-resistant architecture with future WebRTC implementation

## Tech Stack

**Backend:** Django, Django Channels, PostgreSQL  
**Frontend:** Progressive Web App, WebSocket API  
**Security:** End-to-End Encryption planned  
**Protocol:** WebSocket for real-time communication
